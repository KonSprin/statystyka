# Statystyka - model regresji liniowej

Konrad Springer

## Instrukcje
Opracuj model regresji liniowej wielorakiej przewidujący ciśnienie krwi na podstawie zanonimizowanych  wskaźników zdrowia i stylu zycia pacjentów.

### Kryteria oceny:

1. Import danych. Eksploracja danych. omówienie analizy statystycznej danych. Histogramy danych i wnioski z nich płynace. Korelacje danych - omówienie. (20pkt)
2. Wybór zmiennych i budowa modeli (zbadać minimum dwa zestawy danych). Porównanie modeli przy pomocy współczynnika skorygowanego determinacji  (10 pkt)
3. Podanie wzoru modelu, zbadanie istotności współczynników i interpretacje współczynników wybranego modelu oraz interpretacja współczynnika zmienności. (20pkt)
4. Analiza reszt: normalność reszt, test na zerową wartość średniej, homoskedastyczność, autokorelacja (20 pkt)
